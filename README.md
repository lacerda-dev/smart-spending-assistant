# Smart Spending Assistant

Assistente de educação financeira com inteligência artificial, focado em promover maior consciência sobre gastos e incentivar hábitos financeiros mais saudáveis.

Projeto conceitual desenvolvido durante o desafio de IA do Santander Bootcamp 2026 – AI React Front-end, realizado em parceria com a DIO.

## Sobre o projeto

O Smart Spending Assistant é um educador financeiro inteligente criado para ajudar trabalhadores que realizam pequenos gastos impulsivos ao longo do dia e acabam comprometendo suas finanças sem perceber.

A proposta utiliza IA para identificar padrões de consumo, analisar impactos financeiros e sugerir hábitos mais conscientes de economia.

## Objetivos

- Identificar gastos impulsivos recorrentes
- Mostrar o impacto acumulado de pequenas despesas
- Incentivar hábitos financeiros mais saudáveis
- Fornecer recomendações personalizadas
